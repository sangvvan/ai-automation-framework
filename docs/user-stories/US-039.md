# US-039 — Advanced element keywords

## Story
As an **automation tester of complex UIs**
I want keywords for file upload, drag-drop, iframe, and shadow-DOM
So that I can test rich web apps.

## Acceptance criteria

### AC-1: upload_file
- **Given** an `<input type="file">` and a scenario step
  `upload_file(target, '/tmp/foo.png')`
- **When** the step runs
- **Then** the upload completes and the page shows the file name.

### AC-2: drag_drop
- **Given** a sortable list with two items
- **When** a step `drag_drop(item1, item2)` runs
- **Then** items are reordered (verified by next-text assertion).

### AC-3: switch_to_frame
- **Given** an iframe containing a button "OK"
- **When** a scenario uses
  `switch_to_frame({ kind: 'role', role: 'img', name: 'frame' })`
  then `click({ kind: 'role', role: 'button', name: 'OK' })`
- **Then** the click targets the iframe's button.

### AC-4: Shadow-DOM piercing
- **Given** a closed Shadow root containing a button
- **When** a scenario uses
  `click({ kind: 'text', text: 'OK', pierceShadow: true })`
- **Then** the click succeeds.

### AC-5: Keyboard shortcut
- **Given** the new `type_keyboard('Escape')` keyword
- **When** invoked
- **Then** the page receives the keydown event.
